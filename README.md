# 🎯 Focus & Flow — Productivity Dashboard

A sleek, single-file productivity dashboard combining a Pomodoro timer, persistent to-do list, and motivational quote widget — all wrapped in a premium glassmorphism dark-mode UI.

## ✨ Key Features

- ⏱️ **Pomodoro Timer** with Focus (25m), Short Break (5m), and Long Break (15m) modes
- 🎨 **Animated circular progress ring** that visualizes time remaining
- ✅ **To-Do List** with strike-through completion and one-click delete
- 💾 **Persistent storage** — tasks survive page refreshes via `localStorage`
- 💬 **Random motivational quotes** from a curated library, refreshable on demand
- 🔔 **Audio chime** when a session completes (Web Audio API — no external files)
- 📱 **Fully responsive** — beautiful on mobile, tablet, and desktop
- 🌌 **Premium glassmorphism design** with animated gradient background

## 🚀 How to Run

It's a single self-contained file — zero installation, zero dependencies.

1. **Download** the `index.html` file from this repo.
2. **Double-click** the file. It will open in your default browser.
3. That's it. Start focusing.

> Works offline after first load (fonts/icons are cached by your browser).

## 🛠️ Technologies Used

- **HTML5** — Semantic structure and SVG for the circular timer
- **CSS3** — Glassmorphism, gradient animations, Flexbox, CSS Grid, media queries
- **Vanilla JavaScript** — No frameworks, no build step, no `npm install`
- **Web Audio API** — For the completion chime
- **localStorage API** — For persistent to-do storage
- **Google Fonts** (Poppins) & **Font Awesome 6** via CDN
